Two triggers used to rollback in mysql

CREATE DEFINER=`John`@`%` TRIGGER `GSV_DB`.`anchorgroup_BEFORE_INSERT` BEFORE INSERT ON `anchorgroup` FOR EACH ROW BEGIN if (Cluster>=(select MaxCluster from `family` where `family`.`fid` = FID)) then SIGNAL SQLSTATE 'TX000' SET MESSAGE_TEXT = 'Cluster is larger than or equal to MaxCluster record in family'; </br> end if; END

Gitpage+Hexo to set up blog site

Use Gitpage & Hexo with Markdown to create personal blog

  1. Setup Git
  2. Setup Node.js
  3. Markdown Mannual can be found here
  4. Create repository rangerbush.github.io
  5. launch git bash
  6. run

$ npm install -g hexo-cli

  1. check if installation is successfual
    $ hexo version

$ hexo init $ npm install $ hexo g $ hexo s

  1. edit -config.yml with notepad++

deploy: type: git repository: http://github.com/yourname/yourname.github.io.git branch: master
use this if git not found

$ npm install hexo-deployer-git --save
